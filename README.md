# Análisis de Caso: Fundamentos de Machine Learning - DataSolutions S.A.

Este proyecto presenta una solución integral para la toma de decisiones basada en datos en los sectores de **Retail, Streaming y Banca**. 
Se diseñaron flujos de trabajo sistemáticos para abordar problemas de predicción de ventas, detección de abandono de clientes y segmentación de perfiles.

## Contenido del Proyecto
El repositorio contiene el análisis completo desarrollado en un entorno de Python, incluyendo:

* **Caso A (Regresión):** Predicción de ventas semanales mediante Regresión Lineal.
* **Caso B (Clasificación):** Detección de *churn* (abandono) usando Árboles de Decisión.
* **Caso C (Clustering):** Segmentación de clientes bancarios mediante K-Means.

## Tecnologías Utilizadas
* **Lenguaje:** Python 3.13
* **Librerías:** Pandas, NumPy, Scikit-Learn, Matplotlib, Seaborn.
* **Entorno:** Jupyter Notebook / VS Code.

## Resultados Obtenidos
Se incluye un informe técnico en PDF con:
1. Justificación de modelos y análisis de riesgos (Overfitting, Datos Desbalanceados).
2. Métricas de evaluación (MSE, Accuracy y Matrices de Confusión).
3. Diagramas de flujo de trabajo aplicados a casos reales.

## Estructura de Archivos
* `data_solutions.ipynb`: Notebook con el desarrollo del código y visualizaciones.
* `data_solutions.pdf`: Informe ejecutivo final.
* `diagrama_de_flujo.png`: Esquema del flujo de trabajo de ML.
* `requirements.txt`: Dependencias del proyecto.
